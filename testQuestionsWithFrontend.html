Question 1

Reversing Method

You are being given the string 4kpUo49Vbk230criaeYaWpE6LE2BTgK0oR9DfU7JpqM0OGQmPTsOx. Write a program that reverses this string, and then paste it in the answer box.

Example - the string dog would yield the answer god.

Question 2

String Iteration Method

You are being given the string 618850167582094632012348435757910453597532559. Write a program that adds each individual number in this string.

Example - the string 123 would yield the answer 6 because 1+2=3, and 3+3=6.

Question 3

Two parameter String iteration method

Write a program that takes two parameters, a string and a character. This parameter returns the amount of occurences the character appears in the string. The given string is juveosyvvoxslsnnyufmjuchfpbrwiwairhlbydcbpzrivhx and the character is w.

Example - the string ilikeicecream should yield the integer 3 for the number of occurences of the character i.

Question 4

Directions

Imagine you are given a set of directions, where the letter represents the direction, and the number represents the distance:

L4 means you traveled to the left, 4 units.

R3 means you traveled to the left, 3 units.

L9 means you traveled to the left, 9 units.

R2 means you traveled to the left, 2 units.

The thing is, when you travel to the left, it counters the distance you've traveled to the right, and vise-versa. For example, if you are given L7 followed by R4, the total distance traveled is L3.

Example - you are given the directions L4R7L1R3L2R8, the total distance should be R11.

The string you are given is: L5L9L1R5R3L9R2L5R8L9R1L4L4L6R1R5L2R5R2L4L1R8R3L7. Find the total distance traveled.

import java.util.*;
public class Questions {
  // Question 1
    String Reversed(String theString) {
        StringBuilder sb = new StringBuilder();
        ArrayList arrayList = new ArrayList<>();
        for (int i = 0; i < theString.length(); i++) {
            arrayList.add(theString.charAt(i));
        }
        for (int j = arrayList.size() - 1; j >= 0; j--) {
            sb.append(arrayList.get(j));
        }
        return sb.toString();
    }

  // Question 2
    int sumInString(String number) {
      int  sum = 0;
      for (int i = 0; i < number.length(); i++) {
        char currentChar = number.charAt(i);
        int toInt = Character.getNumericValue(currentChar);
            sum += toInt;
        }
        return sum;
    }

  // Question 3
    int numOfCharInString(String theString, char theCharacter) {
      int  runningTally = 0;
      for (int i = 0; i < theString.length(); i++) {
        char match = theString.charAt(i);
        if (match == theCharacter){
            runningTally++;
        }
      }
        return runningTally;
    }

  // Question 4
    String distanceTraveled(String directions){
      int right = 0;
      int left = 0;
      for(int i = 0; i < directions.length(); i++){
        if(i % 2 == 0){
          if(directions.charAt(i) == 'L'){
            char leftChar = directions.charAt(i+1);
            int leftNum = Character.getNumericValue(leftChar);
            left += leftNum;
        }
          else{
            char rightChar = directions.charAt(i+1);
            int rightNum = Character.getNumericValue(rightChar);
            right += rightNum;
          }
          }
      }
      if(left < right){
        return "R"+(right-left);
      }
      else if(left == right) {
        return "There was 0 distance traveled.";
      }
      else{
        return "L"+(left-right);
      }
  }

  }
