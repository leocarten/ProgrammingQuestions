Question 1

Reversing

Question 1 - You are being given the string 4kpUo49Vbk230criaeYaWpE6LE2BTgK0oR9DfU7JpqM0OGQmPTsOx. Write a program that reverses this string, and then paste it in the answer box.

Example - the string dog would yield the answer god.

Question 2

String Iteration

Question 2 - You are being given the string 618850167582094632012348435757910453597532559. Write a program that adds each individual number in this string.
